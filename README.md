# Expression Library

Библиотека для парсинга, вычисления и упрощения математических выражений на языке C.

## 📋 Описание

Библиотека предоставляет функционал для:
- Парсинга инфиксных математических выражений
- Преобразования в постфиксную (обратную польскую) нотацию
- Вычисления выражений с поддержкой арифметических операций
- Подсчёта количества выполненных операций
- Обработки ошибок
